# Meme Editor

## Суть приложения
Веб-приложение для создания goofy ahh мемов. Пользователь может загрузить изображение, создать пустой холст или выбрать гифку из GIPHY, после чего добавлять текст, фигуры и картинки, управлять слоями и настраивать свойства объектов.

## Технологии

**React и Typescript** — база.

**Vite** — сборщик и сервер разработки. Обеспечивает быстрый запуск и горячую замену модулей, что ускоряет разработку по сравнению с Create React App.

**Tailwind CSS** — стилизация. Утилитарный подход позволяет быстро создавать интерфейс без написания кастомного CSS. В отличие от CSS-in-JS, не добавляет лишнего кода в бандл.

**Zustand** — управление состоянием. Хранит ссылку на канвас, настройки инструментов и цвета. Проще Redux, не требует провайдеров и лишнего кода.

**Fabric.js** — работа с холстом. Предоставляет высокоуровневое API для манипуляции объектами на канвасе (добавление, перемещение, изменение свойств). В отличие от Konva.js, лучше подходит для работы с изображениями.

**Axios** — HTTP-клиент для запросов к GIPHY API. Проще встроенного fetch, автоматически обрабатывает JSON и ошибки.

## Общие впечатления
React-router - офигенная тема, очень выручил на этапе, когда страница начала походить на месиво.
FabricJS - на удивление достаточно легкая.
